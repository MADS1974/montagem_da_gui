
🎨 Avaliação 2: Montagem de GUI
Um aplicativo multiplataforma para montagem de interface de cadastro desenvolvido em Flutter e Dart.
O objetivo principal é demonstrar a habilidade de construção de layouts complexos, uso de componentes de entrada de dados e organização de elementos visuais, seguindo as diretrizes de design do framework.


Este projeto foi criado para fins acadêmicos na disciplina de DM1 – DESENVOLVIMENTO MULTIPLATAFORMA 1, com foco em:


Estrutura de arquivos modular e profissional (separação em múltiplos arquivos).

Implementação de TextFields com customização via InputDecoration.

Organização de layout utilizando Columns, Rows e o widget Spacer.

Gerenciamento de entradas de usuário com TextEditingController.

Feedback visual através de Caixas de Diálogo (AlertDialog) informativas.

Validação de formulários para garantir a integridade dos dados inseridos.

🚀 Funcionalidades
✔️ Interface de Cadastro: Layout completo com campos de Nome, Endereço e eMAIL.
✔️ Validação em Tempo Real: Verifica campos obrigatórios e formato de e-mail antes de processar.
✔️ Identidade Visual Customizada: Cores baseadas no padrão oficial do IFSP (Verde e Vermelho).
✔️ Captura de Dados: Ao clicar em salvar, os dados digitados são capturados e exibidos no diálogo.
✔️ Segurança de Layout (SafeArea): Interface adaptada para não ser cortada por entalhes ou barras de sistema.
✔️ Ações de Fluxo: Botões de Cancelar (limpar) e Salvar (processar) com alertas informativos.


🛠 Tecnologias Utilizadas

Flutter — Framework de UI multiplataforma.


Dart — Linguagem de programação.


Android Studio — IDE de desenvolvimento.

Material 3 — Sistema de design moderno.

TextEditingController — Controle e captura de texto.


InputDecoration — Estilização avançada de campos de entrada.

SafeArea — Garantia de visibilidade em diferentes dispositivos.

Form & GlobalKey — Validação estruturada de formulário.
